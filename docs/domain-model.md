# Domänmodell (första utkast)

## Kärnobjekt

- **Idé**: Ett politiskt förslag eller en ståndpunkt.
- **Alternativ**: En annan lösning på samma problem.
- **Argument**: Skäl för eller emot en idé.
- **Källa**: Referens som stärker ett argument.

## Grundrelationer

- En idé kan ha flera alternativ.
- En idé och ett alternativ kan ha flera argument.
- Argument kan kopplas till en eller flera källor.
